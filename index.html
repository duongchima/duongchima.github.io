story

Chi is a recent graduate in George Brown Collegiate's Game Programming program with a passion in building User Interface and gameplay systems. They've had a passion for video games ever since their dad got them a GameBoy Advance where an endless amount of hours were spent playing the Pokemon Mystery Dungeon games. This love continued onto adulthood and inspired Chi to take on the path of being a Game Programmer. Attending George Brown Collegiate, Chi has learned a proficient amount of C++, C#, and can confidently adapt to any engine thrown their way. Even when graduation approaches, they have a thirst for learning and approaches these opportunities with eagerness and preparation to put in the work needed. Video games have always been a key point in Chi's upbringing and they hope they can help build games that will speak to others in the same way.

projects

Listed here are the projects that Chi has worked on throughout the last 3 years.

curtain call crusade, 2023

In the 2023 Otome Jam, Chi was assigned the positon of the lead programmer, mandating tasks and creating the base systems for the game's infrastructure. Main components that they developed was creating a Visual Novel system with the assistance of Ink in the Unity Engine, the base mechanics of the Rhythm Game minigame, and debugging any issues that arose. They were also using tools like DryWetMidi and FMOD for music implementation and creating midi maps for the rhythm game portion. Chi displayed exceptional leadership in their role as a lead programmer as they were able to guide the other programmers in the team and stayed on top of the team's tasks through daily stand-ups and weekly department meetings with the team leads.

monster mystery, 2022

As part of a student developer team at George Brown College, Chi had the opportunity to work on an exciting project to design and implement an online multiplayer system using C++ and Unreal Engine’s network capabilities. To ensure smooth collaboration, they utilized Github’s repository functions and git commands to resolve commit issues. Their focus was also on creating a visually appealing and engaging User Interface that would enhance the multiplayer experience. To complete the project, They was responsible for developing and animating all of the characters.

cells at war, 2022

A collaboration project between McMaster University's Biology Department and George Brown's Design Department, Cells at War's objective is to improve the learning experience in the classroom using tools like video games. Chi was assigned as one of the main programmers in this project, tasked with handling all the UI elements, creating the Encyclopedia using Scriptable Objects, handle the game flow through the GameManager, creating the Expert Deck using LeanTween for the animations, implemented the video cutscenes and loading screens, and animated the character through Unity's animator.

cosmos cafe, 2021

A project done for George Brown's student placement program and Octopus Moon Studios. Chi was assigned to handle the UI elements, building the gameplay mechanic of the Rocket Rascals moving in a line using a List, and allowing the player to click on the objects floating on the screen to add to the tray.

fabric panic, 2021

A prototype done for a student placement program. Chi was assigned to handle the UI elements, gameplay flow, and the drag-and-drop functionality for the objects in the Unpacking Phase.
